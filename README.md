# Parking_lot_management-system-uge-10-11

A Simple parking house management system that can be used to track cars coming and going from parking houses

## Description
This is a RESTful API utilizing a entity frame work to simulate keeping track of parking houses and cars in parking houses. 

## Installation
### Backend
 ```
Run dotnet run .\Parking_lot_management-system-uge-10-11.csproj
 ```
  to start the backend

### Frontend
to run the front end first cd in to the Frontend-Parking-Sytem file then
```
RUN npm install
```
```
run ng serve --open
```
or 
```
run ng serve
```
